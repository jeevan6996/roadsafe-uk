# Product definition

RoadSafe UK is an investigation workspace for comparing observed collision
evidence, exposure-aware statistical screening, and calibrated model outputs at
road-segment level.

## Primary user

The initial user is a road-safety analyst who needs to identify a defensible
shortlist of locations for further investigation. Researchers and community
groups are secondary users.

## Core decision

The product supports deciding which locations warrant closer investigation. It
does not decide which intervention to install and does not claim causality.

## Signature representation

The Model Agreement view will place observed counts, exposure-adjusted rates,
Empirical Bayes estimates, and accepted ML models on a shared road network. It
will emphasize agreement, disagreement, confidence, and data coverage rather
than presenting one opaque risk score.
